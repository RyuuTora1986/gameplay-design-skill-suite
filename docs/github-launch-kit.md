Turn vague game ideas into structured gameplay packages, then expand them into Chinese production-ready specs your AI coding stack can actually execute.

This repo packages an orchestrator-first gameplay design workflow for Codex. Instead of dumping disconnected prompts, it forces direction lock, stage sequencing, scope control, and coding-ready handoff. The result is a more reliable bridge from raw game concept to validated gameplay package and full Chinese design spec.
